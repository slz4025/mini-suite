Shift+Click on a cell or navigate around using Tab (right), Shift+Tab (left), Enter (down), Shift+Enter (up) to change the cell in focus.

This editor allows you to view and manipulate the underlying value or formula of the currently focused cell and update all other cells that depend on this cell. To express the value using a formula, type Python code within angle brackets.

Refer to the index for:

  • current cell's row: <ROW>,
  • current cell's column: <COL>.

Refer to a selection of other cells:

  • single row: <R#r>,
  • multiple rows: <R#rs:re>,
  • single column: <C#c>,
  • multiple columns: <C#cs:ce>,
  • single cell: <R#r><C#c>,
  • box of cells: <R#rs:re><C#cs:ce>,

where

  • r = expression for index of row,
  • c = expression for index of col,
  • rs = expression for inclusive-start index of row range,
  • re = expression for inclusive-end index of row range,
  • cs = expression for inclusive-start index of column range,
  • ce = expression for inclusive-end index of column range.
